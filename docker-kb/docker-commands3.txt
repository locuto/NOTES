Pull an image
docker pull <image name>:<version>
docker pull redis

Create and start a container
docker run --name <container name> -d <image name>:<version>
docker run  --name redis1 -d redis

Restart an exited container - in background
docker start <container name | container id>
docker start redis1

Attach an exited container - to terminal/stdin
docker attach `docker ps -q -l` # reattach the terminal & stdin
docker attach <container name|id> # reattach the terminal & stdin
